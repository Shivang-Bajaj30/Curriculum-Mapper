# Curriculum Mapper

## Setup (everyone run these steps after cloning)

### Backend
1. `cd backend`
2. Create a virtual environment (optional): `python -m venv .venv` then activate it
3. `pip install -r requirements.txt`
4. Copy `.env.example` to `.env` and add your `GEMINI_API_KEY=your_key`
5. Run the backend: `flask run` (keeps running in this terminal)

### Frontend
1. Open a **second** terminal
2. `cd frontend`
3. `npm install`
4. `npm run dev`

### Use the app
- Open http://localhost:5173 in your browser
- Keep **both** the backend terminal and frontend terminal running
